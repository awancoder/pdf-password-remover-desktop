How to Use

Done in 4 easy steps

No technical skills needed. Anyone can use it.

Select PDF Files

Click "Select PDF Files" button or drag & drop one or more PDF files into the available area.

Enter Password

Type your PDF password. Leave it blank if the file has no password but is still protected.

Click Remove Password

Press the "Remove Password" button and let the application process all files automatically one by one.

Done!

Files with removed passwords are automatically saved in the same folder with _unprotected.pdf name.
